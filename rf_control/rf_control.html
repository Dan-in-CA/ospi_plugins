RF Control

This is a work in progress. It is for testing and development only.

There are 2 ways to add cli commands to this plugin:

  • Edit the file SIP/plugins/rf_control.py lines 77 and 85. These commands use the number provided as the var 'i' to set the device.
  • Edit the file SIP/data/rf_control.json. There are separate commands for each station and for on and off. Be careful to keep the file valid JSON. You will also need to comment out lines 77 and 85 in rf_control.py and un-comment lines 76 and 84.
