Weather_level_adj Plugin Documentation

About the Plugin

The Weather_level_adj plugin can be used to control irrigation times based on weather information and/or prevent irrigation during cold conditions

Using the plugin

A couple of the option fields on the plugin page must be filled in order for the plugin to function.


First, you will need a free OpenWeather API key in order for the plugin to be able to get weather data. You can get an API key by creating a free account on the Openweathermap web site.

Enter your API key into the appropriate field on the plugin page.


The location field can contain a city name or postal code. However, the most accurate weather data will be obtained by using geographic coordinates (latitude - longitude) in the location field.

You will need to enter your coordinates as decimal degrees in the following format: 

lat=XXX.XXX_lon=XXX.XXX

Note the underscore between the lat and lon values and do not include any spaces.


If you do not know your geographic coordinates, an easy way to find them is by using Google Maps.
On your computer search for your location on the map then right click your location and select What's here? from the menu that appears. At the bottom, you will see a card with your coordinates.


Once you have the Location and API key fields filled in you need to check one or both of the check boxes at the top of the page depending on how you want the plugin to function.


After clicking the Submit button to save your settings the plugin should be operational and you can adjust the remaining options to suit your needs.


Note:
If you change the Preferred Units option, any other changes made at the same time will be ignored. Only the type of units displayed will be changed.


The Max percentage field has a maximum possible value of 200.


The History days used and Forecast days used fields have maximum values of 5.


The Daily irrigation field should hold an estimated amount of water your system applies on a per-day basis. If, like most users, you don't irrigate every day, divide the total estimated amount of water your system applies in a week by 7 to get the average daily amount. An internet search should list a number of sites that provide information abut how to measure and/or estimate the amount of water an irrigation system applies.


The Status box is used by the plugin to display information about it's operation. It may display error messages but if the plugin is operating properly it will show information it is using to adjust the irrigation time. If there is no information in the box, it may take up to an hour for the plugin to acquire data to display. If you leave the plugin page open you will need to refresh the page to see updated information.
The current temperature is updated hourly and can be compared to another source such as an online weather report for your area to determine if the plugin is getting accurate data. The Temperature shown by the plugin probably won't be an exact match to another source but it should be within a few degrees.

Notes:

This plugin uses a set of weight values to adjust watering times. You can modify the values used by editing the PrecipWeights section of the weather_decipher.json file in SIP's data directory.


After the plugin is installed it will take a few days for it to accumulate the necessary historical data. Open Weather Map does not supply historical data for free.
