combine_stations Plugin Documentation

About the combine_stations plugin

This plugin allows multiple stations to be run at the same time (concurrently) when SIP is in sequential mode.

The stations to be combined are grouped into up to 8 virtual stations which can then be added to an irrigation program, run in manual mode or run from the Run Once page.

The plugin uses the final 8 stations from SIP's list of stations.


NOTE: if you are using relay boards to control your sprinkler valves be sure the power supply for the relays can support multiple relays being on at the same time.

It is not advised to run multiple relays which are powered only from the Raspberry Pi. A seperate power supply for the relays is recommended

Using the combine_stations plugin

Before configuring any virtual stations It is necessary to add 8 additional stations to SIP's user interface (UI). You can do this by opening the OPTIONS page and going to Station handling. Increase the Station extensions setting by one. This will add 8 stations to SIP's UI.

These additional stations can then be used as virtual stations that combine actual system stations to operate as one.


For each virtual station, enter numbers, separated by commas, for the stations to be combined.

For example, a station list might look like: 3,6,7


After one or more virtual stations are configured, and the settings saved, the plugin renames the virtual stations which then start with "V" followed by the station number, the word "runs" and the list of combined stations.

The virtual stations can then be added to your irrigation programs with the combined stations shown.
