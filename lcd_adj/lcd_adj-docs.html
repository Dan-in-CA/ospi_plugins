LCD Plugin Documentation

About the LCD Plugin

The LCD plugin is designed to display SIP status information on a 16x2 character LCD display. The display is connected with the I2C bus using a PCF8574.

Each Item will show on the display for 4 seconds.

Compatible with HD44780 LCD 16x2 controller.

Using the LCD Plugin

Use I2C LCD

Cause the plugin to start displaying information on the LCD

I2C Address

Select the I2C address for your device

SIP Software Version

Show the SIP software version

IP

Show the IP address of the device SIP is running on

Port

Show the port number of the web server

CPU Temp

Show the CPU temperature of the device SIP is running on

Time and Date

Show the current time and date

Uptime

Show the system uptime

Rain Sensor

Show the rain sensor status

Running Stations

Show list of running stations

Status

Show LCD plugin status

Troubleshooting

The most likely cause of information not showing on the display is having the incorrect I2C address selected for your hardware. Use i2cdetect to check the address of your device.

sudo i2cdetect -y 1

If you use one of the first RPI (Raspberry Pi Model B 256 megabytes), then we have to change the command:

sudo i2cdetect -y 0

Should the LCD not work when connected, Use I2C LCD is checked and the correct I2C address is selected then it is probably best to run SIP from the command line as the LCD plugin prints status information.

If using a Raspberry Pi, and possibly other hardware, the 5v GPIO is active on power up so the back light may come on but nothing will display on the LCD if the settings are incorrect or the wiring of the I2C bus is incorrect.
