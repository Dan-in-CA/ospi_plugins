telegramBot Plugin Documentation

About the telegramBot Plugin

The telegramBot plugin enables SIP to communicate the change of status via a telegram (www.telegram.org) bot.

The plugin includes the folowing files:

  • telegramBot.py
    The python program that does the actual work of the plugin. Location: SIP/plugins

  • telegramBot.html
    A web.py template that provides a user interface for the plugin. It includes both Python code and web page markup. Location: SIP/templates

  • telegramBot-docs.html
    This file. Provides documentation for the plugin including any special hardware needed and set up instructions. Location: SIP/static/docs/plugins

  • telegramBot.manifest
    A simple text file containing a list of each of the plugin’s files with the path to it’s location. The manifest is used by the plugin manager to install and uninstall the plugin. The file can also contain information about the plugin such as author, brief description, requirements, etc. Location: SIP/plugins/manifest

  • telegramBot.json
    An optional file used for persistant storage of plugin settings. It is created by the plugin the first time it is run and is not installed with the other files. This allows for software updates without overwriting existing settings. Location: SIP/data

Using the telegramBot plugin

As a first step you must install the Telagram API python modules (python-telegram-bot):

  1. Log-in to your PI as root

  2. Install the libraries as using:

    # apt-get install python-pip
# pip install python-telegram-bot

  3. Reboot your PI

Then you must provide a “token” to enable the bot to talk to telegram, follow the instructions Here

This plugin is disabled by default. If you are reading this, the telegramBot plugin is probably enabled on your system. Otherwise you can enable it using the plugin manager or by setting the group permissions of the telegramBot.py file to executable. Once it is enabled, and the sip program has been restarted, You can verify it is working by selecting it from the PLUGINS drop down menu of the main program’s web interface. You should be taken to the plugin’s Settings page.
